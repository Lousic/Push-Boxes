# Push Boxes

1.0
简单介绍一下我做的推箱子游戏,比较简单也没什么好说的=_=
因为一开始有几节课请假,所以一上来就跟不上讲的进度了,就在CDSN上找了个大佬的代码参考,加上自己的理解创新,写出了这个简单的推箱子程序
自认为还算创新的点:
1,没搞懂EGE,就做的控制台版本的,不过游戏界面看起来比学长的更好看一点,用了一些特殊符号
2,因为没参考学长的代码,所以代码应该不同的地方还是挺多的,代码查重率低也算是创新点吧=_=
3,实现了跳关功能,打了比较多的注释,还是比较好看懂得
